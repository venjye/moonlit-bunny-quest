# 月塔归兔

全屏网页小游戏，玩法按经典魔塔设计：走格子、捡钥匙、开门、看伤害、攒属性、算路线。主角改成小兔露娜，故事换成沿月塔追回姐姐与归家月铃。

## 运行

在仓库根目录执行：

```bash
python3 -m http.server 8080 --bind 0.0.0.0
```

打开：

- `http://127.0.0.1:8080/bunny_adventure/`
- `http://你的局域网IP:8080/bunny_adventure/`

## 模式

- `故事模式`：12 层主线，逐层揭开露娜姐姐为什么留在塔顶
- `冒险模式`：100 层长线爬塔，更接近传统魔塔的资源规划
- `无尽模式`：自动续层，直到角色倒下

## 操作

- 桌面端：方向键或 `WASD`
- 手机 / 平板：底部触控键
- `查`：读取相邻目标与战斗预判
- 撞上怪物自动按魔塔公式结算

## 配置

主要配置在 `game-config.json`：

- 标题、钩子、随机种子
- 初始属性与钥匙
- 三种模式的层数
- 剧情段落、敌人模板、商人模板、场景色板

## 玩法骨架

- 固定 13x13 格
- 黄 / 蓝 / 红门钥匙体系
- 敌人按可预估公式战斗
- 商人用金币换生命 / 攻击 / 防御
- 故事模式加入剧情节点，但核心玩法仍是魔塔式路线计算
- 右侧面板内置怪物手册，可直接看当前层敌人的数值与预估伤害
- 支持浏览器本地存档 / 读档，以及失败 / 通关结算面板
